# Streaming Audio
## A simple app to test streaming audio and video URLs for compatibility with iOS's AVFoundation

Find a URL that points to a streaming audio or video source. Type or paste it into the URL text field. Press the play button.

Here are some URLs that are known to work:
- [Apple Developer Test Stream](http://devimages.apple.com/iphone/samples/bipbop/bipbopall.m3u8)
- [Developing Perspective Podcast](http://static.developingperspective.com/developing_perspective_212.mp3)
